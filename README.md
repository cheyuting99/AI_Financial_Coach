# AI Financial Coach for Students

An intelligent multi-agent financial coaching platform designed specifically for college students.

---

##  Overview

AI Financial Coach is a personalized financial guidance system that helps college students:

-  Track spending habits  
-  Manage debt strategically  
-  Start investing with confidence  
-  Receive AI-powered financial coaching  

Instead of just tracking numbers, our platform analyzes financial behavior and provides actionable, personalized advice.

---

##  The Problem

Many college students:

- Overspend without realizing it  
- Accumulate credit card or student debt  
- Feel overwhelmed by investing  
- Lack access to personalized financial education  

Most budgeting apps focus on tracking — not coaching.

---

##  Our Solution

We built a **multi-agent AI financial system** that acts as a real-time personal financial coach.

The system:

- Understands the user’s full financial profile  
- Detects spending patterns and financial risks  
- Suggests customized repayment strategies  
- Provides beginner-friendly investment guidance  
- Teaches financial concepts in plain language

##  System Architecture

### 1️⃣ Frontend

A user-friendly dashboard with five main pages:

- Overview  
- Spending  
- Debt  
- Invest  
- AI Chat  

Users interact through visual analytics and natural language questions.

---

### 2️⃣ Backend

- API layer processes financial data  
- Structured dataset includes:
  - Transactions
  - Income
  - Debt
  - Savings
  - Investment preferences  

---

### 3️⃣Multi-agents Architecture
#### Super Agent (Orchestrator)

The Super Agent:

- Interprets user queries  
- Identifies financial intent  
- Routes requests to relevant sub-agents  
- Aggregates responses into a coherent final answer  


#### Sub Agents

#####  Spending Insights Agent
- Categorizes expenses  
- Detects overspending trends  
- Recommends behavioral adjustments  

#####  Debt Strategy Agent
- Calculates payoff simulations  
- Recommends avalanche vs snowball method  
- Projects repayment timelines  

#####  Investment Coach Agent
- Evaluates risk tolerance  
- Suggests beginner-friendly investment strategies  
- Explains financial concepts step-by-step  

---

## Agent Workflow

1. User submits a question (e.g., "Should I start investing now?")
2. Super Agent analyzes the intent
3. Relevant sub-agent(s) are activated
4. Financial data is processed
5. Super Agent synthesizes insights
6. User receives a personalized response
